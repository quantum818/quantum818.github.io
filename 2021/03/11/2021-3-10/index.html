在Vscode上配置Java

发表于2021-03-10|更新于2021-03-11
|阅读量:

＂封面是牛哥最喜欢的秋山澪＂

引言

　　牛哥以前拿我用VS写C++小程序开过玩笑，而且总是以能使用二刺猿插件作为原因向我推荐 vscode， 之后牛哥就勾搭上了同校的欧洲金发女留学生，也很少再和我说这件事。这次开学我也正打算先把java给学起来，尝试了一下在vscode上配置java，发现效果不错。

过程

  • 首先在vscode上安装以下的插件
    • Java Extension Pack
    • Java Test Runner
    • Language Support for Java(TM) by Red Hat
    • Project Manager for Java
    • Maven for Java
    • Debugger for Java
  • 然后在AdoptOpen 上下载JDK(我下载的版本是java SE11，另外在vscode上安装插件时也会跳出JDK的下载，选择一种即可)
  • 这些都完成后在vscode设置中搜索Java.home，选择在settings.json中编辑来添加JDK的路径。

测试

全部配置完后在vscode中按下Ctrl+Shift+P选择Java:Creat Java Project... No build tools来创建一个项目并输入


点击debug终端输出何励晗傻逼则表示完成配置。
